### Vue工程添加favicon.ico



- create a favicon image, like favicon.png

- add it to static

- declear fav in index.html

  ```html
  <head>
      <meta charset="utf-8">
      <link rel="shortcut icon" type="image/png" href="/static/favicon.png"/>
      <title>My Vue.js app</title>
      ...
  </head>
  ```
