## Window OS 下java路径

Windows 操作系统和大多数其他操作系统一样， 对文件系统做了树形规划，每个节点可能是文件也可能是文件夹；java程序在windows操作系统下引用文件路径的时候，可以使用2种方式：

-   方式1 - 类Linux方式

    /I:/github/java/java-core/src/main/java/com/jim/java/core/chapter1/

-   方式2 - Windows转义方式

    ```shell
    i:\\github\\java\\java-core\\src\\main\\java\\com\\jim\\java\\core\\chapter1
    ```
